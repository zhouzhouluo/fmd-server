        郑州博凯医药保健品有限公司成立于2000年4月，注册资金5000万元，位于国家郑州高新技术产业开发区，是专门从事保健食品、 保健用品、功能食品、消杀用品等的研发、生产、销售、服务于一体的高科技企业。公司以技术创新带动快速发展，目前净资产已过亿元。 公司先后通过了保健食品GMP、功能食品QS、消杀用品GMP、化妆品生产许可证、出口食品认证，在同行业内是国内最早、河南唯一的同时拥有药监、 卫生、技术监督、进出口商品检验检疫四大最高权威认证的企业。

        公司为河南省保健协会常务副会长单位，是世界泡腾制剂保健品生产的龙头企业，公司在2013年获得河南省商务厅等17个厅局（委）联合命名的         “河南省十大诚信经营典范单位”荣誉称号，产品先后被评为中国保健品公信力产品奖、河南名牌产品。VC泡腾片、康伊丽牌大豆异黄酮泡腾片是 极具特色的政、商公务接待及私人馈赠的礼品，成为郑州市乃至河南省的一张名片。

为了满足国内外市场快速增长的需求，公司正在建设的年产30亿片泡腾制剂及系列保健食品、消杀品生产线项目，总投资3.52亿，占地150亩， 是河南省“十二五”重点建设项目，并获得河南省2010年特色产业基金80万元的资金支持。

        公司拥有较强的自主开发能力，与国内外多家科研机构及高校建立了合作关系。公司注重技术开发及自有知识产权的保护，拥有国家发明专利、 外观保护专利多项。公司拥有泡腾片、颗粒冲剂、咀嚼片剂、口服液、胶囊、膏剂、丸剂、贴剂、洗液等多种剂型的生产能力及技术，是国内 拥有自主知识产权最多、品种最全、保健食品批文最多的泡腾片专业的生产厂家。2011年8月，国内首家泡腾制剂研究中心—郑州泡腾制剂工程 研究中心通过了郑州市发改委的评审，并获得郑州市政府30万元的奖励。

        “以人为本 务实创新 诚信感恩”是公司的核心理念，“做令人尊敬的健康顾问，造世界一流的健康产品，创全球知名的中国品牌” 是博凯不懈的追求。16年来，我们一直在努力！
